# Debian Zsh 配置与标签显示脚本

一个为Debian延伸版设计的Zsh配置脚本，提供现代化终端体验和自定义标签显示功能。

## ✨ 核心功能

### 🎨 终端美化
- 自动安装并配置Zsh
- 操作系统智能着色（Debian/Ubuntu/Kali等不同颜色）
- Root用户红色警示提示符
- 增强的命令补全和历史记录

### 🏷️ 标签显示系统
- 显示三行自定义标签信息
- 支持到期提醒（可选）或永久标识
- 简洁直观的服务器状态展示

### 🔒 一键配置
- 自动检测并适配Debian衍生系统
- 智能处理用户权限和Shell切换
- 完整的备份和恢复机制

## 📊 显示示例

```bash
[Debian GNU/Linux 13 (trixie)] 2C/1GB
到期: 永久 (∞) • 甲骨文 • 东京 • NTT
```



## 🚀 快速开始

```bash
# 下载并运行脚本
curl -O https://example.com/VPS-Zsh-Dashboard.sh
sudo bash VPS-Zsh-Dashboard.sh
```
